# CodeTracker Frontend

React frontend for CodeTracker, a coding progress tracker built with Java, Spring Boot, MySQL, REST APIs, and React.

## Tech Stack

- React.js
- JavaScript
- React Router
- Axios
- Chart.js / react-chartjs-2
- React Icons
- React Datepicker
- React Hot Toast
- Plain CSS

## Available Scripts

Run the frontend locally:

```powershell
npm start
```

Build the production bundle:

```powershell
npm run build
```

## Pages

- Login
- Register
- Dashboard
- Coding Sessions
- Goals
- Problems
- Reports
- Achievements
- Profile
